# Customer Info Project

Простое Java-приложение для демонстрации работы с объектами, инкапсуляцией и выводом данных в консоль.

## Функционал
Программа собирает данные о клиенте (имя и номер телефона), создает объект класса `Customer` и выводит отформатированную информацию в консоль.

## Структура проекта
Проект организован следующим образом:

src/
└── app/
├── Customer.java # Модель данных клиента (имя, телефон)
└── Main.java # Точка входа в программу и логика обработки

## Требования
*   **Java SDK** (версия 8 или выше)
*   **IntelliJ IDEA** (или любая другая Java IDE)

## Как запустить
1.  Склонируйте или создайте проект в вашей IDE.
2.  Убедитесь, что файлы находятся в пакете `app`.
3.  Откройте файл `Main.java`.
4.  Нажмите правую кнопку мыши и выберите **Run 'Main.main()'**.

## Пример вывода
После запуска вы увидите в консоли:
`Customer: Tom, phone 555 123-8596`

## Особенности реализации
*   В классе `Customer` использованы модификаторы `private final` для обеспечения неизменяемости (immutability) данных.
*   Доступ к полям осуществляется через публичные методы `getName()` и `getPhone()`.
